# House Price Prediction

## Overview

This project was developed as part of **Internship Project – Week 1**. It uses machine learning techniques to predict residential house prices based on property characteristics and housing amenities.

## Dataset

* **Source:** Kaggle – Housing Prices Dataset
* **Records:** 545 residential properties
* **Target Variable:** `price`

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Models Implemented

* Linear Regression
* Random Forest Regressor

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score
